# Drafts

All auto-generated drafts land here. The daily session (Phase H) writes
draft replies and communications to this directory. Each draft includes
TO, SUBJECT, CONTEXT, and DRAFT BODY.

Drafts are NEVER auto-sent. The operator reviews and sends manually.
