# Communication Style

- **Language:** British English
- **Currency:** AED/USD (never other currencies without conversion)
- **Emojis:** No emojis in any output
- **Tone:** Professional, concise
- **Format:** Prefers options + tradeoffs over single-path recommendations
- **Output length:** Concise — say it in fewer words, not more
- **Drafts:** Always provide context + rationale, never just the body
